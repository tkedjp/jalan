# jalan
じゃらんのサイトのスクレイピングコードです。
スクレイピングした情報をCSVかスプレッドシートとwordに保存する仕様です。

それぞれMacとWindowsのGUIアプリも作りました。
CSVとwordファイルは実行ファイルと同じディレクトリに保存されます。
